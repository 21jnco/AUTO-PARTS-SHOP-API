# Auto Parts Shop API

Auto Parts Shop API is a backend project for a small auto parts store.

The goal of this project is to practice building a REST API with multiple related database models, CRUD operations, foreign keys, relationships, and simple order business logic.

## Tech Stack

- Python
- FastAPI
- MySQL
- SQLAlchemy ORM
- Pydantic
- Docker
- Git / GitHub

## Project Status

This project is currently in development.

## Main Features

Planned MVP features:

- Manage product categories
- Manage products
- Manage customers
- Create customer orders
- Add multiple products to one order
- Calculate total order price
- Decrease product stock after order creation

## Main Entities

- Category
- Product
- Customer
- Order
- OrderItem

## Purpose

This is a learning project created to improve backend development skills with FastAPI, SQLAlchemy, MySQL, Docker, and REST API architecture.
